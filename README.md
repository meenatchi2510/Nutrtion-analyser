## вҡҷпёҸ How It Works

1. User enters or selects food items.
2. The application recognizes the food.
3. Nutritional information is retrieved.
4. Calories, protein, carbohydrates, and fats are calculated.
5. Charts visualize nutritional intake.
6. AI recommends healthier food choices.
7. Daily nutrition history is tracked.
